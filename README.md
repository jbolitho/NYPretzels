# NYPretzels
Just a test of GitHub Pages using a pretzel recipe.

## Preview / Run locally

This repository contains a single static HTML file: [pretzel_recipe.html](pretzel_recipe.html).

- Open directly in your browser (double-click the file), or from a terminal:

```bash
xdg-open pretzel_recipe.html
```

- Serve over a local HTTP server (recommended when testing JavaScript):

```bash
# Python 3 (serves the current directory on port 8000)
python3 -m http.server 8000
# then open http://localhost:8000/pretzel_recipe.html
```

- Alternative: use `npx http-server` or the VS Code Live Server extension:

```bash
npx http-server -p 8000
```

## Publish

To publish on GitHub Pages, enable Pages in the repository settings and select the `main` branch (root) or deploy from a `gh-pages` branch.
